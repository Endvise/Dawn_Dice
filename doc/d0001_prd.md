## 1. 개요

### 1.1 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 0001_vibe |
| 목적 | AI 에이전트 기반 코딩 환경 템플릿 |
| 유형 | Type C (에이전트/CLI 도구 프로젝트) |

### 1.2 프로젝트 설명

Vibe 코딩 환경은 Claude Code, Gemini 등 AI 에이전트를 활용한 개발 워크플로우를 정의하는 템플릿 프로젝트입니다.

**핵심 구성요소**:
- `.claude/` - Claude Code 설정 및 SuperClaude 프레임워크
- `v/` - oais 스킬 문서 및 가이드
- `CLAUDE.md` - 에이전트 진입점
